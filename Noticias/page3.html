Bem-vindo à terceira página!

Agora, se quisermos voltar para a primeira página, você deve estar pensando que é só colocar o link aqui como fizemos anteriormente, porém não é. A lógica seria colocar o link direto como index.html, mas assim como foi diferente para o link da terceira página, tambám vai ser diferente para voltarmos ao index, pois a página 3 e o index não estão na mesma pasta.

Devemos então ao digitar o link na tag a, antes do arquivo index.html, devemos colocar ../

Isso significa que estamos pedindo ao "código" para voltar uma pasta. De acordo com a lógica do LINUX, pois os nossos servidores estão guardados no LINUX.

O código vai ficar assim:

< a href="../index.html">primeira página< /a>
